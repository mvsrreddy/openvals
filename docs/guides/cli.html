CLI Usage

Command-line interface for rapid model benchmarking and evaluation.

Commands

version

Displays the current version and build information.

openvals version

datasets

Lists all built-in datasets available for benchmarking.

openvals datasets

benchmark

The primary command for running side-by-side model comparisons.

openvals benchmark [OPTIONS]

Options

Option Description Required
--dataset Name of the built-in dataset to use (e.g., finance). Yes
--models Comma-separated list of Ollama model names. Yes
--config Name of a configuration preset (e.g., healthcare). No
--output Filename for the HTML report (default: report.html). No
--report / --no-report Whether to generate an HTML report (default: --report). No

Examples

Compare two local models on the finance dataset:

openvals benchmark --dataset finance --models llama3,mistral

Run a code evaluation with specialized weights:

openvals benchmark --dataset dev --models codellama --config developer
