API Reference / Recommendation

Recommendation Engine

Analyzes benchmark results to suggest the best model based on use-case profiles.

python 
from openvals.recommendation.engine import RecommendationEngine

RecommendationEngine Class

class RecommendationEngine(results)

Parameters

Parameter Type Description
results Dict The dictionary of results returned by BenchmarkRunner.run().

Methods

recommend(use_case="default")

Generates a model recommendation based on a specific use-case profile.

Parameter Type Default Description
use_case str "default" The profile to use. Options include "accuracy", "speed", "balanced".

Returns: A dictionary containing:

  • recommended_model: The name of the top-ranked model.
  • score: The recommendation score.
  • confidence: A confidence score (Score × DRS).
  • reason: A human-readable explanation for the choice.
  • tradeoffs: Analysis of what is sacrificed vs alternatives.
  • risks: List of potential flags (e.g., "High Latency").

Example Usage

python 
from openvals.recommendation.engine import RecommendationEngine

# Initialize with benchmark results
engine = RecommendationEngine(benchmark_results)

# Get recommendation for high-accuracy needs
rec = engine.recommend(use_case="accuracy")

print(f"Recommended: {rec['recommended_model']}")
print(f"Reason: {rec['reason']}")
