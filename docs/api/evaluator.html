API Reference / Evaluator

Evaluator

The core class for running AI model evaluations against datasets.

python 
from openvals.core.evaluator import Evaluator

Evaluator Class

class Evaluator(model, dataset, weights=None, debug=False)

Parameters

Parameter Type Default Description
model BaseModel Required An instance of a class that inherits from BaseModel.
dataset List[Dict] Required A list of samples containing input and expected_output keys.
weights Dict[str, float] See description The weight assigned to each metric. Default: accuracy: 0.35, semantic: 0.30, latency: 0.15, reliability: 0.10, safety: 0.10.
debug bool False If True, prints detailed per-sample evaluation results to the console.

Task-Specific Logic

The Evaluator applies different logic based on the type key found in your dataset samples:

  • classification: The semantic_similarity score is forced to match the accuracy score.
  • generation: If the output exceeds 120 words, a 10% penalty is applied to the semantic_similarity score.
  • default: Standard evaluation logic applies.

Semantic Thresholds

You can define a semantic_threshold in the evaluation dictionary of a sample. If the semantic score meets or exceeds this threshold, it is automatically rounded up to 1.0.

Methods

run()

Executes the evaluation pipeline for the provided model and dataset.

Returns Description
Dict A dictionary containing:
  • overall_score: Weighted average score.
  • drs_score: Dynamic Response Score.
  • metrics: Averages for all metrics.
  • samples: Detailed results per input.

Example Usage

python 
evaluator = Evaluator(
    model=my_llm,
    dataset=my_data,
    weights={"accuracy": 0.8, "semantic": 0.2, "latency": 0.0}
)

results = evaluator.run()
print(f"Score: {results['overall_score']}")
