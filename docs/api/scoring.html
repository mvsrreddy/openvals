API Reference / Scoring

Scoring

Methods for calculating final model performance scores.

weighted_score(metrics, weights)

Calculates the final trust score by summing the product of each metric and its corresponding weight.

from openvals.scoring.weighted import weighted_score

metrics = {"accuracy": 0.9, "semantic": 0.8, "latency": 0.1}
weights = {"accuracy": 0.4, "semantic": 0.4, "latency": 0.2}

final_score = weighted_score(metrics, weights)

compute_drs(metrics)

Calculates the Dynamic Response Score, which measures the naturalness and generation quality of the model output.

from openvals.scoring.drs import compute_drs

drs = compute_drs(avg_metrics)

Default Weights

If no weights are provided to the Evaluator, the following defaults are used:

Metric Weight
Accuracy 0.35
Semantic Similarity 0.30
Latency 0.15
Reliability 0.10
Safety 0.10
