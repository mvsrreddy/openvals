API Reference / Metrics

Metrics

Built-in evaluation metrics for measuring AI model performance.

accuracy(pred, expected)

Calculates exact string match accuracy (normalized to 0.0 or 1.0).

from openvals.metrics.accuracy import accuracy

score = accuracy("Hello world", "Hello world") # 1.0
score = accuracy("Hello world", "Hi world")    # 0.0

reliability_score(model, prompt)

Measures the stability of a model's output. It executes the same prompt 3 times and calculates the average semantic similarity between all resulting pairs.

from openvals.metrics.reliability import reliability_score

score = reliability_score(model, "What is the capital of France?")

consistency(model, prompt)

Measures the exact-match stability of outputs. It executes the prompt 3 times and returns 1.0 / count(unique_outputs).

from openvals.metrics.consistency import consistency

score = consistency(model, "Yes or No?") # 1.0 if all responses same

semantic_score(a, b)

Calculates the meaning-based similarity between two strings using the all-MiniLM-L6-v2 sentence transformer model.

from openvals.metrics.semantic import semantic_score

score = semantic_score("The car is fast", "The vehicle is quick")

embedding_distance(a, b)

Calculates the vector distance between two strings using pre-trained embeddings (requires sentence-transformers).

from openvals.metrics.embeddings import embedding_distance

score = embedding_distance("AI is future", "Artificial Intelligence is the next step")

fuzzy_similarity(a, b)

Calculates a fuzzy matching score between two strings using Levenshtein distance.

from openvals.metrics.similarity import fuzzy_similarity

score = fuzzy_similarity("OpenVals", "Open Vals") # 0.9

measure_latency(func, *args, **kwargs)

A utility wrapper to measure the execution time of a function.

from openvals.metrics.latency import measure_latency

result, seconds = measure_latency(my_model.generate, "What is AI?")

reliability(output)

Evaluates the structural stability and formatting consistency of the model output.

safety(output)

Detects potentially harmful, biased, or unsafe content in the generated text.

consistency(model, input)

Measures how stable the model's output is across multiple runs for the same input.

variance(outputs)

Calculates the statistical variance across a set of model responses.

verbosity(output)

Analyzes the length and word density of the output relative to the task type.

format_compliance(output, expected)

Checks if the output follows the expected length constraints based on the length of the expected output.

from openvals.metrics.format import format_compliance

score = format_compliance("Short answer.", "Short answer.") # 1.0

Metric Interpretation Guide

Use this guide to understand the scores and their production readiness.

Metric Ideal Direction Good Range Meaning
Accuracy Higher ↑ 0.80 → 1.00 Correctness of output
Semantic Higher ↑ 0.75 → 1.00 Meaning similarity and contextual alignment
Reliability Higher ↑ 0.70 → 1.00 Stability across repeated generations
Safety Higher ↑ 0.85 → 1.00 Lower risk and harmful behavior
Consistency Higher ↑ 0.75 → 1.00 Repeatability of model behavior
Variance Lower ↓ 0.00 → 0.25 Output deviation across runs
Latency Lower ↓ 0ms → 1500ms Response generation speed
