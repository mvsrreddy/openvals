API Reference / Benchmarking

Benchmarking

Classes and utilities for side-by-side model comparison.

BenchmarkRunner

Manages the execution of multiple models against a single dataset.

from openvals.benchmarking.benchmark import BenchmarkRunner

models = {"gpt-4": model_a, "claude-3": model_b}
runner = BenchmarkRunner(models, dataset)
results = runner.run()

Normalization

Converts metric values into a 0.0 to 1.0 range based on the minimum and maximum values found across all models in the benchmark.

from openvals.benchmarking.normalization import normalize_scores

normalized = normalize_scores(results)

Ranking

Produces a sorted list of models based on their weighted normalized scores.

from openvals.benchmarking.ranking import rank_models

weights = {"accuracy": 0.5, "semantic": 0.3, "latency": 0.2}
ranking = rank_models(normalized, weights)
