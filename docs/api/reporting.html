API Reference / Reporting

Reporting

Generate visual reports and summaries of your model evaluations.

python 
from openvals.reporting.html_report import generate_html_report

generate_html_report()

Creates a standalone HTML file containing a model leaderboard and recommendation summary.

def generate_html_report(results, recommendation, output_file="report.html")

Parameters

Parameter Type Default Description
results Dict Required Benchmark results from BenchmarkRunner.
recommendation Dict Required Recommendation data from RecommendationEngine.
output_file str "report.html" The name of the file to save.

Example Usage

python 
from openvals.reporting.html_report import generate_html_report

# 1. Run benchmark
results = runner.run()

# 2. Get recommendation
recommendation = engine.recommend()

# 3. Generate report
generate_html_report(results, recommendation, "my_report.html")
