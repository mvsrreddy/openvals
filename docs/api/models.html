API Reference / Models

Models

Interfaces and implementations for AI model adapters.

BaseModel

The abstract base class that all model adapters must inherit from.

from openvals.models.base import BaseModel

class BaseModel(ABC):
    @abstractmethod
    def generate(self, prompt: str) -> str:
        """Returns the model response for a given prompt."""
        pass

DummyModel

A reference implementation used for testing. It simply reverses the input string.

from openvals.models.dummy_model import DummyModel

model = DummyModel()
response = model.generate("hello") # returns "olleh"

OllamaModel

Native adapter for running local models via Ollama.

from openvals.models.ollama_model import OllamaModel

model = OllamaModel(model_name="llama3")
response = model.generate("What is AI?")
