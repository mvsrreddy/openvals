API Reference / Datasets

Datasets

Utilities for loading and managing evaluation datasets.

Built-in Datasets

OpenVals includes several curated datasets for different industry domains. You can list them using the CLI:

openvals datasets

load_builtin_dataset(name)

Loads a built-in dataset by its registered name.

from openvals.datasets.loader import load_builtin_dataset

dataset = load_builtin_dataset("finance")

load_dataset(path)

Loads a JSON dataset from a local file path.

from openvals.datasets.loader import load_dataset

dataset = load_dataset("my_data.json")

Expected Format

json 
[
  {
    "id": "optional_unique_id",
    "input": "The prompt to send to the model",
    "expected_output": "The reference answer"
  }
]

Manual Datasets

You can also provide a list of dictionaries directly to the Evaluator without using the loader.

dataset = [
    {"input": "Hi", "expected_output": "Hello"},
    {"input": "2+2", "expected_output": "4"}
]

Metadata Schema

Each built-in dataset includes a dataset.json file that defines its metadata and recommended evaluation settings.

json 
{
  "name": "Finance Advanced",
  "domain": "finance",
  "version": "1.0",
  "description": "Financial reasoning evaluation",
  "recommended_metrics": ["accuracy", "safety", "reliability"],
  "recommended_weights": {
    "accuracy": 0.4,
    "safety": 0.4,
    "reliability": 0.2
  }
}
