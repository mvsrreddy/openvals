Getting Started

Set up OpenVals and run your first AI model evaluation in minutes.

Installation

OpenVals is available on PyPI. You can install it using pip:

bash 
pip install openvals

For development or the latest features, you can install from source:

bash 
git clone https://github.com/vishwanathakuthota/openvals.git
cd openvals
pip install -e .

Dependencies like numpy, pandas, and scikit-learn will be installed automatically.

Your First Dataset

OpenVals expects evaluation data in a simple JSON format. Create a file named my_eval.json:

json 
[
  {
    "id": "1",
    "input": "What is 2+2?",
    "expected_output": "4"
  },
  {
    "id": "2",
    "input": "Translate 'Hello' to Spanish",
    "expected_output": "Hola"
  }
]

Running via CLI

The quickest way to run an evaluation is using the built-in CLI command:

bash 
openvals run --dataset my_eval.json

This will use the DummyModel (which reverses input) to demonstrate the evaluation flow. You'll see an output containing the overall trust score and metric averages.

Running via Python

For deeper integration, use the Python API. This allows you to use your own model adapters.

python 
from openvals.core.evaluator import Evaluator
from openvals.models.dummy_model import DummyModel
from openvals.datasets.loader import load_dataset

# 1. Load data
dataset = load_dataset("my_eval.json")

# 2. Setup model
model = DummyModel()

# 3. Initialize Evaluator
evaluator = Evaluator(model, dataset)

# 4. Run and see results
results = evaluator.run()
print(results)

Local Models with Ollama

OpenVals now natively supports local models via Ollama. This is ideal for private or air-gapped evaluations.

python 
from openvals.models.ollama_model import OllamaModel

# Initialize with any model pulled in Ollama
model = OllamaModel("llama3")

# Use it in the same evaluator flow
evaluator = Evaluator(model, dataset)
result = evaluator.run()
