Why OpenVals?

Most AI evaluation tools stop at simple metrics. OpenVals goes further by aligning evaluation with your specific business objectives. We provide the tools to quantify trust, risk, and performance in a way that translates directly to deployment confidence.

Key Capabilities

  • Multi-Model Benchmarking: Side-by-side comparison under identical conditions (Ollama + APIs).
  • Weighted Scoring: Trust Score engine with customizable weights for accuracy, cost, and latency.
  • Expanded Metric Suite: Reliability, Safety, Consistency, and Variance checks.
  • Recommendation Layer: Suggests the best model and provides automated tradeoff analysis.

Quick Example

python 
from openvals.core.evaluator import Evaluator
from openvals.models.ollama_model import OllamaModel
from openvals.datasets.loader import load_dataset

# Load dataset and local model
dataset = load_dataset("examples/sample_eval.json")
model = OllamaModel("llama3")

# Run evaluation with Trust Scoring
evaluator = Evaluator(model, dataset)
result = evaluator.run()

print(f"Overall Trust Score: {result['overall_score']}")
print(f"DRS Score: {result['drs_score']}")
