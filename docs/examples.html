Examples

Real-world usage patterns for OpenVals.

Basic Evaluation

Using the Python API to evaluate a local model.

python 
from openvals.core.evaluator import Evaluator
from openvals.models.dummy_model import DummyModel

dataset = [
    {"input": "reverse this", "expected_output": "siht esrever"}
]

model = DummyModel()
evaluator = Evaluator(model, dataset)
result = evaluator.run()

print(f"Final Score: {result['overall_score']}")

Benchmarking Multiple Models

Compare two different models side-by-side.

python 
from openvals.benchmarking.benchmark import BenchmarkRunner
from openvals.benchmarking.normalization import normalize_scores
from openvals.benchmarking.ranking import rank_models

models = {
    "model_a": MyModelA(),
    "model_b": MyModelB()
}

runner = BenchmarkRunner(models, dataset)
results = runner.run()

normalized = normalize_scores(results)
ranking = rank_models(normalized, {"accuracy": 0.5, "latency": 0.5})

for name, score in ranking:
    print(f"{name}: {score:.2f}")

CLI Benchmarking

Running a benchmark from the terminal.

openvals benchmark --dataset examples/sample_eval.json

Local Models with Ollama

Evaluate a model running locally via Ollama.

python 
from openvals.models.ollama_model import OllamaModel
from openvals.core.evaluator import Evaluator

model = OllamaModel(model_name="llama3")
evaluator = Evaluator(model, dataset)
print(evaluator.run())

Model Recommendation

Get a weighted recommendation based on benchmark results.

python 
from openvals.recommendation.engine import RecommendationEngine

# Assuming 'results' is from BenchmarkRunner.run()
engine = RecommendationEngine(results)
rec = engine.recommend(use_case="customer_support")

print(f"Recommended Model: {rec['recommended_model']}")
print(f"Confidence: {rec['confidence']:.2f}")
