Roadmap

Our vision and upcoming features for OpenVals.

v0.1.5 (Current)
  • Core evaluator and Multi-model benchmarking.
  • DRS Scoring and Recommendation Engine.
  • HTML Reporting and Configurable profiles.
  • Ollama integration and Typer CLI workflows.
v0.2.0 (In Progress)
  • Advanced normalization algorithms.
  • Dataset expansion and synthetic data tools.
  • Embedding-powered semantic scoring (Redesign).
  • Reliability and Safety scoring redesign.
v0.3.0
  • Explainability layer (SHAP/LIME integration).
  • REST API for remote evaluation.
  • Evaluation history and trend tracking.
  • External dataset integrations (HuggingFace).
v1.0.0
  • Full SaaS Platform.
  • Enterprise Governance and Compliance tools.
  • Continuous AI validation pipelines.
  • Team workspaces and collaborative dashboards.

Vision

OpenVals is evolving into a Trust Layer for AI Systems. We want to provide a definitive answer to questions like:

  • Which model should be deployed for this specific task?
  • Is the model safe and aligned with business goals?
  • Can the model be trusted in a production environment?
